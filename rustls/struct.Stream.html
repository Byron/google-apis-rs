pub struct Stream<'a, S: 'a + Session + ?Sized, T: 'a + Read + Write + ?Sized> {
    pub sess: &'a mut S,
    pub sock: &'a mut T,
}

This type implements io::Read and io::Write, encapsulating a Session S and an underlying transport T, such as a socket.

This allows you to use a rustls Session like a normal stream.
